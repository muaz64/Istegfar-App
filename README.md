# Istighfar Reminder (ইস্তেগফার রিমাইন্ডার)

A modern, user-friendly Progressive Web Application (PWA) designed to help users maintain their daily Dhikr and Istighfar. This app is specifically optimized for Bangladesh with localized timings and calendar support.

## Key Features

- **Interactive Tasbih Counter:** A large, accessible counting circle with haptic feedback (vibration) to keep track of your Dhikr without looking at the screen.
- **Prayer Timetable:** Accurate prayer times specifically calculated for Chattogram, Bangladesh.
- **Ramadan Dashboard:** Dedicated section for daily Sehri and Iftar timings with a real-time live countdown.
- **Dynamic Calendar:** Dual display showing both the Gregorian (English) date and the Hijri (Arabic) date, adjusted for the Bangladeshi moon sighting.
- **Smart Reminders:** Set custom intervals or a specific daily time to receive push notifications for Dhikr.
- **PWA (Offline Support):** Can be installed on the home screen of any mobile device. Once loaded, it works perfectly without an internet connection.

## Technologies Used

- **HTML5 and CSS3:** For a fully responsive, mobile-first UI with smooth animations.
- **JavaScript (ES6+):** Handles prayer time calculations, countdown logic, and notification scheduling.
- **Service Worker:** Manages caching for offline functionality and seamless background updates.
- **Web App Manifest:** Enables the Add to Home Screen feature for a native app-like experience.

## How to Use

1. Open the web app link in your browser.
2. Click the Save App (অ্যাপটি সেভ করুন) button to install it on your mobile device.
3. Tap the central Tasbih Counter to start your Zikir.
4. Use the settings cards to set your preferred reminder intervals or specific prayer alerts.
